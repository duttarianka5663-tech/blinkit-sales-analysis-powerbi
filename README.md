## Project Title
Blinkit Sales Analysis - Power BI Dashboard

## Objective
To analyze Blinkit outlet sales performance across 
locations, outlet types, and product categories.

## Tools Used
Power BI, DAX, Data Visualization

## Key Findings
- Total sales of $1.20M across 8,523 items analyzed
- Supermarket Type 1 outlets generated highest revenue at $787.55K
- Tier 3 locations contributed maximum sales at 472.13K
- Fruits and Snack foods were top selling item categories
- Sales peaked in 2018 at $205K before declining

## Dataset
Blinkit grocery outlet sales dataset with 
product, location, and outlet type information
